# Single Page Curriculum Vitae

https://roadmap.sh/projects/single-page-cv

Simple CV website showcasing skills, experience, and projects.

## How to Run

1. Clone the repository:
   git clone https://github.com/marvini-louz/single-page-cv.git

2. Navigate to the project folder:
   cd your-repository

3. Open the project:
    - Open `index.html` in your browser
