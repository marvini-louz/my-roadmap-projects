# Basic HTML Website

## Project Overview

In this project, I created a simple component for a website that displays a changelog. The component serves as a structured log of all notable changes made to a project or software, helping users stay informed about updates, improvements, and fixes.

## Getting Started

To view the website, simply open the HTML files in a web browser. You can navigate between pages using the navigation bar.

### Local Development

1. Clone or download the repository to your local machine.
2. Open any of the `.html` files in your preferred web browser to view the site.
